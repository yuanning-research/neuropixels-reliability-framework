# Neuropixels Cross-day Matching
Ongoing research project on cross-day neuron matching and validation in chronic Neuropixels recordings.

## Goal
Track putative neurons across recording sessions and evaluate the reliability of matching outputs.

## Current Work
- Cross-session matching of sorted units
- Waveform comparison across days
- Auto-correlogram analysis from spike times
- Review of high-confidence matched pairs

## Current Findings
- Identified candidate cross-day neuron pairs with high matching scores
- Observed both agreement and mismatch patterns across waveform and firing signatures
- Built an initial framework for larger-scale validation

## Planned Next Steps
- Apply the pipeline to larger datasets
- Develop automated validation metrics
- Extend tracking across multiple sessions

## Tools
MATLAB, Python, neural data analysis
